## Usage

```
C:.
├─bin
├─config
├─models
├─public
│  ├─images
│  ├─javascripts
│  └─stylesheets
├─routes
└─views
```

- config = 각종 개인 설정 파일
- models = 데이터 모델 설정 파일
- public = static 파일
- routes = 라우팅



1. Install modules 
```
npm install

or

yarn
```

2. Modify the config file

Fix config file 

3. Run server
```
npm start 

or 

yarn start
```
